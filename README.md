# 개발자 포트폴리오

검정색과 보라색을 주로 사용한 모던한 개발자 포트폴리오 웹사이트입니다.

## 기술 스택

- HTML5
- CSS3
- JavaScript (Vanilla)

## 주요 기능

- 반응형 디자인 (모바일, 태블릿, 데스크톱 지원)
- 부드러운 스크롤 애니메이션
- 프로젝트 포트폴리오 섹션
- 기술 스택 소개
- Discord 웹훅을 통한 연락 폼

## 프로젝트

1. **TodayToYou** - AI 기반 감정 일기 및 타로 카드 웹 애플리케이션
2. **WiFi 보안 체커** - WiFi 네트워크 보안 모니터링 Windows 애플리케이션
3. **고객 피드백 분석 대시보드** - Streamlit 기반 데이터 분석 도구

## 배포

이 프로젝트는 GitHub Pages를 통해 배포됩니다.
